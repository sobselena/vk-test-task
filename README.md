# 🚀 Запуск проекта

### 1. Клонировать репозиторий

``` bash
git clone https://github.com/sobselena/vk-test-task.git
```

### 2. Перейти в папку проекта

``` bash
cd vk-test-task
```

### 3. Установить зависимости

``` bash
npm install
```
### 4. Скопировать переменную окружения в .env
``` bash
  cp .env.default .env
```
### 5. Запустить проект

``` bash
npm run dev
```

После запуска приложение будет доступно по адресу:

    http://localhost:5173
